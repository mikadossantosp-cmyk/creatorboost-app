01 / Was ist das hier

🎯 Unsere Mission

CreatorX ist eine geschlossene Creator-Community, in der sich Mitglieder gegenseitig auf Instagram pushen — durch echte Likes, echte Kommentare und echtes Engagement. Kein Bot-Verhalten, keine künstlichen Wachstumsversprechen.

  • Jeder hilft jedem — Engagement gegen Engagement
  • Natürliches Verhalten, kein Auto-Liker
  • Nachhaltiges Wachstum durch echte Interaktion
  • Punkte (XP) für aktives Mitwirken, Belohnungen via Diamanten
💡 Warum gibt es Regeln? Damit das System für alle fair bleibt und Instagram die Aktivität nicht als Manipulation erkennt.

03 / Web-App

🔍

Web-Push-Benachrichtigungen bei Likes, Kommentaren oder DMs (opt-in)

PWA + APK

04 / Punkte-System

📈 XP & Badges

Du sammelst XP durch Aktivität in der Community. Mit XP steigst du in Levels und Badges auf — höhere Badges schalten Vorteile frei.

XP verdienen

Link posten+1 XP
Like geben+5 XP
Daily Login Bonus+10 bis +30 XP
Mission 1 (5 Links liken)+5 XP
Mission 2 (80% liken)+5 XP
Mission 3 (100% liken)+5 XP + 💎
Weekly Streak M2 (7×)+15 XP + 💎
Weekly Streak M3 (7×)+20 XP + 2 💎

📐 Level-Formel: Level = ⌊XP / 100⌋ + 1

Badge-Stufen

🆕 New0 — 49 XP
📘 Anfänger50 — 499 XP
⬆️ Aufsteiger500 — 999 XP
🏅 Erfahrener1.000 — 4.999 XP +1 Link/Tag
👑 Elite5.000 — 9.999 XP +1 Link + Mo-Bonus
🌟 Elite+ab 10.000 XP 2 Superlinks/Woche
🏆 Trophäen: Bei jedem Badge-Aufstieg bekommst du eine Trophäe in dein Profil. Trophäen bleiben für immer.
05 / Tägliche Aufgaben

🎯 Missionen

📅 Daily Missionen — Auswertung 12:00 Uhr

M1 — 5 Links liken+5 XP
M2 — 80% aller Links liken+5 XP
M3 — 100% aller Links liken+5 XP + 💎
⚠️ Wer einen Link postet, ohne mindestens M1 zu erfüllen, kassiert eine Verwarnung.

📆 Weekly Streaks — Reset Montag 00:05 Uhr

M1 Streak — 7 Tage M1+10 XP
M2 Streak — 7 Tage M2+15 XP + 1 💎
M3 Streak — 7 Tage M3+20 XP + 2 💎
💡 Warum Streaks? Engagement-Streaks belohnen Mitglieder, die kontinuierlich aktiv sind — nicht nur einmal die Woche.
06 / Full Engagement

🌟 Superlinks

Ein Superlink ist ein Premium-Posting im Full-Engagement-Thread. Wer einen Superlink postet, kriegt von allen anderen Mitgliedern Likes, Kommentare, Saves & Shares — die volle Reichweiten-Power der Community.

  • 1 Superlink pro Woche für alle (Elite+: 2 pro Woche)
  • Nur Montag bis Samstag möglich (Sonntag ist Auswertungstag)
  • Instagram-Profil per /setinsta Pflicht
  • Wer einen Superlink postet, MUSS alle anderen Superlinks der Woche liken
  • Erinnerung Sonntag 21:00 · Auswertung Sonntag 23:59

Verstöße & Konsequenzen

Engagement-Pflicht verletzt−50 XP + ⚠️
Posten ohne /setinsta❌ blockiert
Mehrfach pro Woche❌ blockiert
💡 Wie posten? Mit /superlink im Bot oder über den +-Button in der App.

08 / Spielregeln

📋 Regeln & Verwarnungen

🔗 Link-Regeln

1 Link pro Tag (außer Bonus-Links)❌ blockiert
Nur Instagram-Links❌ blockiert
Keine Duplikate⚠️ Warnung
Kein Self-Like⚠️ Warnung
Sperrzeit Sonntag 20:00 — Montag 06:00❌ blockiert
Links älter als 48 Std.🗑️ Auto-Löschung

✅ Erlaubte Link-Formate

  • Posts: instagram.com/p/...
  • Reels: instagram.com/reel/...
  • Profile: instagram.com/dein.profil

❌ Nicht erlaubt

  • Story-Links (laufen nach 24h ab)
  • TikTok, YouTube, Threads, andere Plattformen
  • Verkürzte Links (bit.ly, t.co, ...)

🤝 Respekt & Umgang

  • Sei respektvoll zu jedem Mitglied
  • Kein Hate, keine Beleidigungen
  • Keine Diskriminierung (Herkunft, Geschlecht, Religion, etc.)
  • Kein Spam in Chats
  • Keine Werbung für andere Plattformen oder Bots
  • Konstruktives Feedback statt Kritik
  • Bei Konflikten: Admin per DM
  • Was im Chat passiert, bleibt im Chat

⚠️ Verwarnungs-System

1 / 5Hinweis
2 / 5Hinweis
3 / 5Letzte Chance
4 / 5Kritisch
5 / 5🔨 Auto-Ban
🎁 Warn-Abbau: 5 Tage M1 in Folge → 1 Verwarnung wird gelöscht.
09 / App auf dem Handy

📲 App installieren

Android (Chrome)

  1. App-URL in Chrome öffnen
  2. Auf den roten 📱 App installieren-Button unten rechts tippen
  3. Im Chrome-Dialog "Installieren" wählen
  4. Fertig — App liegt auf dem Homescreen

iOS (Safari)

  1. App-URL in Safari öffnen
  2. Auf das Teilen-Symbol tippen (Quadrat mit Pfeil)
  3. "Zum Home-Bildschirm" wählen
  4. "Hinzufügen" bestätigen

Android (APK)

  1. APK aus Telegram-Channel herunterladen
  2. Bei Erstinstallation einmalig "Apps aus unbekannten Quellen" erlauben
  3. APK öffnen → installieren → fertig
  4. Updates kommen automatisch beim nächsten App-Öffnen
⚡ Auto-Update: Die App lädt Inhalte live vom Server. Bei jedem Öffnen siehst du den neuesten Stand. Bei Bugs: App komplett schließen → neu öffnen.
Letzter Tipp

🚀 Erster Tag

  1. Bot starten mit /start
  2. Instagram verbinden per /setinsta deinprofil
  3. Profilbild & Spitzname in der App setzen
  4. Ersten Link posten
  5. Mindestens 5 andere Links liken → Mission 1 erfüllt
🎉 Damit hast du am ersten Tag schon ~25 XP + Daily-Bonus + die ersten Trophäen.
